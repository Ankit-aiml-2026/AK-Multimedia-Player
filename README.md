# AK-Multimedia-Player

This is my second project, built using AI-assisted development tools. It took around 2 weeks to complete.

## ✨ Key Features

- 🎵 Separate Audio & Video playlists
- ⌨️ Full keyboard-controlled playback system
- 📱 Fully responsive across PC, Tablet, and Mobile
- 💾 Automatically restores previously loaded playlists after revisiting the website
- 🔄 Seamless switching between audio and video modes without reloading playlists
- 🌊 Smooth transitions between songs with wave-like effects
- 🎚️ Built-in audio engine with:
  - Bass controls
  - Mid controls
  - Treble controls
  - Reverb effects
  - Playback speed controls

## ▶️ Additional Features

- ⏭️ Auto-next playback
- 📍 Seekable progress bar
- 📂 Local folder-based media loading
- 💾 Persistent playlist memory using browser storage
- 🎛️ Smooth media control workflow

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Browser Local Storage
- Web Audio API
